| # | 支柱 | 含义 |
|---|------|------|
| 1 | **羁绊感** | 玩家与6名角色产生情感连接，不只是数值操作 |

## 二、游戏概述

### 世界观
喵星是一个人类与猫族共存的魔法大陆。一股神秘的"虚无之雾"正在侵蚀各地，吞噬生命与记忆。  
勇士**臻宝**与法师**李小宝**受命出征，一路结识猫族伙伴，共同寻找传说中的「星之水晶」以拯救喵星。

### 核心角色

#### 🧍 人类主角（固定出战）
| 角色 | 职业 | 性格 | 核心技能定位 |
|------|------|------|-------------|
| 臻宝 | 战士 | 勇敢坚毅，是团队主心骨 | 高物理输出 + 控制减速 |
| 李小宝 | 法师 | 聪明好奇，团队智囊 | 群体魔法攻击 + 元素弱点利用 |

#### 🐱 初始猫咪伙伴（剧情解锁，固定4只）
| 角色 | 职业 | 性格 | 核心技能定位 |
|------|------|------|-------------|
| 艾米 | 辅助/治疗师 | 温柔善良，森林中被救下 | 单体/群体治疗 + 复活 |
| 安妮 | 法师 | 冷静高冷，来自魔法塔 | 高爆发单体魔攻 + 沉默 |
| 钱多多 | 战士 | 搞笑忠诚，爱财重情义 | 嘲讽 + 反伤 + 金币掉落加成 |
| 小贝 | 坦克 | 憨厚可靠，高大威猛 | 超高防御 + 护盾 + 格挡 |

#### 🐾 可收集猫咪（后期扩展）
- 探索地图发现野生猫咪，战斗/道具/支线任务可招募
- 上限：背包可携带12只，出战席位固定6人
- 每只可收集猫咪有独特性格台词，强化羁绊感

## 三、核心游戏循环

### 即时循环（0-30秒）
```
玩家操作 → 选择技能/目标 → 动画演出 → 即时反馈（数字飘字/音效/屏幕震动）→ 下一个角色行动
```
- **行动**：回合轮到某角色，玩家选择「普攻 / 技能 / 道具 / 防御」
- **反馈**：命中特效 + 伤害数字 + 角色语音一句话
- **奖励**：击杀奖励飘出（金币/经验/物品图标）

### 会话循环（5-30分钟）
```
进入地图 → 探索节点 → 触发事件（战斗/宝箱/NPC/隐藏猫咪）→ 战斗胜利 → 获取资源 → 角色成长 → 解锁新区域
```
- **目标**：完成当前地图的主线节点或支线任务
- **张力**：HP/MP有限，无法无限探索；遭遇强敌需要准备
- **解决**：战斗胜利推进剧情，失败保留进度可重试

### 长期循环（天-周）
```
主线剧情推进 → 解锁新地图/新角色 → 养成深度（等级/装备/技能树）→ 收集图鉴（猫咪/道具/怪物）→ 日常任务奖励
```
- **进度**：主线章节解锁，猫咪图鉴填充
- **留存钩**：每日签到奖励 + 限时猫咪招募活动 + 排行榜

## 四、战斗系统详细设计

### 4.1 基础战斗流程
```
战前准备 → 行动顺序计算 → 玩家回合（选择行动）→ 敌人回合（AI行动）→ 状态结算 → 循环至战斗结束
```

### 4.2 行动顺序
- 由**速度(SPD)**属性决定行动顺序
- 每个角色/敌人的行动槽满了才能行动（类似 FFX ATB）
- 速度展示在战斗界面上方时间轴

### 4.3 技能系统

每个角色拥有 **1个普攻 + 3个技能 + 1个必杀技**：

| 技能类型 | 说明 | 消耗 |
|---------|------|------|
| 普攻 | 单体攻击，无消耗 | 0 MP |
| 技能1 | 低消耗，高频使用 | 10-20 MP |
| 技能2 | 中消耗，有附加效果 | 30-40 MP |
| 技能3 | 高消耗，强力效果 | 50-60 MP |
| 必杀技 | 充能触发，华丽演出 | 必杀槽满触发 |

**必杀槽充能方式：** 受到伤害 / 造成伤害 / 特定状态触发

### 4.4 联动技能（核心差异化）
> 设计目标：让玩家思考阵容搭配，而非无脑堆伤害

| 触发条件 | 联动效果 | 涉及角色 |
|---------|---------|---------|
| 臻宝「击晕」目标后，安妮对同一目标施法 | 伤害×1.8，追加「崩裂」效果 | 臻宝 + 安妮 |
| 艾米治疗后，受治疗角色的下次攻击 | 附加「神圣」属性，额外20%伤害 | 艾米 + 任意角色 |
| 小贝使用护盾后，钱多多立即追加一次攻击 | 「守护反击」，对攻击小贝的敌人造成150%反击 | 小贝 + 钱多多 |
| 李小宝「水系」魔法命中后，安妮「电系」魔法命中同一目标 | 触发「导电」，麻痹目标1回合 | 李小宝 + 安妮 |

### 4.5 属性克制系统
```
火 → 冰（克制）→ 雷（克制）→ 风（克制）→ 火（循环）
光 ↔ 暗（互克）
无属性：不受克制影响
```

### 4.6 状态效果
| 状态 | 效果 | 持续 |
|------|------|------|
| 中毒 | 每回合损失5%最大HP | 3回合 |
| 燃烧 | 每回合损失8%最大HP，防御-20% | 2回合 |
| 麻痹 | 50%概率跳过行动 | 2回合 |
| 石化 | 无法行动，受到物理伤害+50% | 1回合 |
| 沉默 | 无法使用技能 | 2回合 |
| 护盾 | 吸收一定伤害 | 至护盾值耗尽 |
| 鼓舞 | 攻击+20%，速度+10% | 3回合 |

## 五、角色成长系统

### 5.1 等级系统
- 等级上限：**Lv.60**（MVP阶段先做到Lv.30）
- 升级方式：战斗获取经验值（EXP）
- 每级提升：HP / 攻击 / 防御 / 速度 按职业比例成长

### 5.2 经验曲线（[占位符] 待调整）
```
目标曲线：前10级平均20分钟/级，10-30级平均1小时/级，30-60级每级需显著资源投入
EXP需求公式（草稿）：EXP(n) = 100 × n^1.5
```

| 等级 | 累计EXP需求 | 单局战斗EXP | 预估升级时间 |
|------|-----------|------------|------------|
| 1→5 | 1,000 | 50-80 | ~15分钟 |
| 5→10 | 5,000 | 80-120 | ~30分钟 |
| 10→20 | 30,000 | 120-200 | ~2小时 |
| 20→30 | 120,000 | 200-350 | ~5小时 |

> ⚠️ 所有数值标记为 **[占位符]**，需游戏测试后调整

### 5.3 装备系统
- 每个角色可装备：武器 + 防具 + 饰品（共3槽）
- 装备品质：普通（白）→ 精良（绿）→ 稀有（蓝）→ 史诗（紫）→ 传说（橙）
- 获取途径：战斗掉落 / 商店购买 / 合成

### 5.4 技能树
- 每个角色有独立技能树（3条分支）
- 消耗「技能点」解锁（升级获得）
- 分支方向对应不同流派，鼓励重复培养不同角色

### 6.1 地图结构
```
世界地图（章节选择）
└── 章节地图（区域探索）
    ├── 普通节点（战斗/宝箱/NPC）
```

## 七、猫咪收集系统

### 7.1 收集机制
- **野生遭遇**：特定地图节点随机遭遇流浪猫咪
- **招募条件**：战斗胜利后满足条件（HP>50% / 特定道具 / 好感度）
- **好感度系统**：每次战斗后台词互动 + 赠送喜爱道具提升好感
- **好感度奖励**：满好感解锁专属技能 / 皮肤 / 隐藏剧情

### 7.2 猫咪图鉴
- 共设计 **30只**可收集猫咪（MVP版先做10只）
- 图鉴收集进度在主界面展示，强化收集驱动力
- 稀有猫咪有专属动画和语音

### 7.3 猫咪稀有度
| 稀有度 | 数量 | 获取方式 |
|--------|------|---------|
| ⭐ 普通 | 10只 | 主线地图随机遭遇 |
| ⭐⭐ 稀有 | 10只 | 特定章节/条件解锁 |
| ⭐⭐⭐ 史诗 | 7只 | 支线任务/隐藏地图 |
| ⭐⭐⭐⭐ 传说 | 3只 | 限时活动/特殊成就 |

- 位置：主界面底部 / 地图浏览界面底部

### 8.3 变现预估（[占位符] 假设数据）
```
假设：DAU = 5,000 / eCPM = 30元 / 人均日看广告次数 = 3次
日收益估算 = 5,000 × 3 × (30/1000) = 450元/天
月收益估算 ≈ 13,500元
```

## 十、技术规格

### 10.1 平台与引擎
- **平台**：微信小游戏
- **引擎**：Cocos Creator 3.x
- **语言**：TypeScript
- **分辨率**：设计分辨率 750×1334（竖屏）

### 10.2 性能目标
- 主流中端机稳定 **60 FPS**
- 包体大小 **< 20MB**（微信小游戏限制）
- 首次加载时间 **< 5秒**

### 10.3 微信小游戏审核策略
> 个人开发者无法发布"游戏"类目，采用以下策略：

- **类目申报**：工具 / 教育（定位为"猫咪故事互动应用"）
- **关键词规避**：审核材料避免出现"战斗""击杀""血量"等词
- **界面调整**：血量条改为"精力值"，攻击改为"互动"
- **内容基调**：强调"探索故事""猫咪收集""人猫友情"主题

### 10.4 美术风格建议
- **风格**：2D 像素风（省美术成本，工具可用Aseprite）或 Q版手绘（AI生成辅助）
- **色调**：暖色系，柔和明亮，符合猫咪主题
- **UI参考**：勇者斗恶龙系列（复古回合制UI）

## 十三、角色技能详细设计

> 设计原则：每角色有明确且不重复的战术位置；技能有联动钩子；数值全标 `[占位符]`

---

### 13.1 臻宝（战士）— 「钢铁先锋」

**战术定位：** 前排物理输出 + 控制节奏 + 联动触发器  
**玩家幻想：** 我是一把锋利的剑，第一个冲上去，用力量压制一切

| 技能 | 名称 | 类型 | MP消耗 | 效果 | 调整杠杆 |
|------|------|------|--------|------|---------|
| 普攻 | 利剑突刺 | 单体物理 | 0 | 造成 `[110%]` 物理伤害 | 系数 |
| 技能1 | 横扫千军 | 全体物理 | 20 | 对所有敌人造成 `[80%]` 物理伤害 | 系数/MP |
| 技能2 | 震地击 | 单体物理 | 35 | 造成 `[150%]` 物理伤害，`[70%]` 概率使目标**击晕**1回合 | 击晕概率/系数 |
| 技能3 | 破甲斩 | 单体物理 | 50 | 造成 `[200%]` 物理伤害，目标防御永久降低 `[15%]`（可叠加3层） | 降防幅度/层数 |
| 必杀技 | 星之剑刃 | 单体物理 | 必杀槽 | 对单体造成 `[400%]` 物理伤害，无视 `[30%]` 防御，击退效果（目标本回合不能行动） | 系数/忽视防御% |

> 🔗 **联动触发**：「震地击」击晕目标 → 安妮跟进施法 → 伤害×1.8 + 「崩裂」效果

---

### 13.2 李小宝（法师）— 「元素智者」

**战术定位：** 群体魔法输出 + 元素弱点利用 + 联动引导者  
**玩家幻想：** 我掌握自然的力量，一场魔法可以改变整个战局

| 技能 | 名称 | 类型 | MP消耗 | 效果 | 调整杠杆 |
|------|------|------|--------|------|---------|
| 普攻 | 魔力弹 | 单体魔法(无属性) | 0 | 造成 `[100%]` 魔法伤害 | 系数 |
| 技能1 | 水流涌动 | 单体魔法(水) | 20 | 造成 `[120%]` 水属性魔法伤害，目标速度 -`[20%]`，持续2回合 | 减速幅度/持续 |

## 十四、敌人设计 — 第一章：翠林迷途

> **设计原则：** 前期敌人用来教玩家规则，不是用来卡关的。每种敌人至少有一个「可读的行为模式」，让玩家能预判并产生策略。

### 14.1 梯队一：新手引导怪（节点1-3）

#### 黑雾菇菇
- **外形：** 被黑雾覆盖的蘑菇，头顶渗着紫色液体
- **数值（Lv.1-2）：** HP `[80]` / ATK `[8]` / DEF `[5]` / SPD `[6]`

- **弱点：** 火属性（×1.5）
- **掉落：** 金币`[10-20]` / 草药×1（低概率）
- **设计意图：** 引导玩家第一次体验「中毒」状态和艾米的净化技能

#### 迷失松鼠
- **外形：** 眼睛通红、毛发凌乱的迷失松鼠

- **行为：** 急速撕咬`[90%]`ATK，本回合行动**2次**；HP<30%时速度额外+`[50%]`
- **弱点：** 无（教玩家关注速度时间轴）
- **掉落：** 金币`[8-15]` / 松果（收集道具）
- **设计意图：** 让玩家第一次理解速度影响行动顺序
